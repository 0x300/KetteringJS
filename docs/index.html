Overview

KetteringJS is a library meant to give easy and structured access to various information and functions through Javascript.

Getting started

Download the latest library file generated in the bin directory from Github. jQuery is the only dependency. Import the file into your project and enjoy!

If you want to modify or build the project yourself, use the MAKE file. The source files will be concatenated and minified into the bin folder. Python and minifier are required dependencies.
