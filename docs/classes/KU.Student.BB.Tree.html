KU.Student.BB.Tree

Module: KU

Holds all the folders and files within a single layer of a content tree for a course.

Item Index

Attributes

  • contentId
  • folders

bbId

String

Course id for Blackboard

contentId

String optional

Content id for root folder on blackboard (when available)

Array

Folders found inside the blackboard folder! Type is KU.Student.BB.Folder[];
